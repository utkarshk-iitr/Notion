DOM Manipulation

DOM (Document Object Model) is a programming interface for HTML and XML documents. It represents the structure of a document as a tree of objects, allowing developers to manipulate the content, structure, and style of a web page.

DOM Manipulation refers to the process of modifying the structure, content, or styling of an HTML document using a programming language such as JavaScript. It's a crucial aspect of dynamic web development, enabling developers to create interactive and responsive web pages.

Let’s first create a sample HTML file - 

<!DOCTYPE html>
<html lang="en">
<head>
    <title>DOM Manipulation</title>
    <script src="script.js" defer></script>
</head>
<body>
</body>
</html>

Here script.js is a JavaScript file while we will use to for DOM Manipulation. Initially it is empty.

Adding Elements

To add elements to DOM we first need a container which will refer to our desired area. Then we can add an element to it using append .

const body = document.body
body.append("Hello India")

We can also use it append multipe data at a time, like body.append(”a”,”b”,”c”) .

We also have another method appendChild but it takes only one parameter at a time, so it is less preferred.

Creating Elements

We can create element and add it to the DOM in a similar way as shown - 

const div = document.createElement('div')
body.append(div)

body.appendChild() can also be used here.

const st = document.createElement('strong')
body.appendChild(st)

Modifying Element Text

There are mainly three methods to modify the element in DOM - innerText , innerHTML and textContent.

const div = document.createElement('div')
div.innerText = "Hello IITR" 
div.textContent = "Hello IITR Again" 
div.innerHTML = "<strong>InfosecIITR</strong>"
body.append(div)

The main difference between innerText and textContent lies when we have a hidden element in HTML and we call them in console -

Changing HTML to - 

<!DOCTYPE html>
<html lang="en">
<head>
    <title>DOM Manipulation</title>
    <script src="script.js" defer></script>
</head>
<body>
    <div>
        <span>Hello</span>
        <span style="display: none;">Bye-invisible</span>
    </div>
</body>
</html>
const dq = document.querySelector("div")
console.log(dq.innerText)
console.log(dq.textContent)

We see that innerText shows only visible text in the console whereas textContent shows all the text in it along with the indentation.

However we should avoid using innerHTML as it is vulnerable to JS injection , can be used to get sensitive data.

Removing Elements

We can also give ID to our tags like - 

<!DOCTYPE html>
<html lang="en">
<head>
    <title>DOM Manipulation</title>
    <script src="script.js" defer></script>
</head>
<body>
    <div>
        <span id="a" title="1">Hello</span>
        <span id="b" title="2">Bye</span>
    </div>
</body>
</html>

Then we select the element by their ID and we can remove/add them - 

const div = document.querySelector("div")
const span1 = document.querySelector("#a")
const span2 = document.querySelector("#b")

span2.remove()
div.append(span2)
div.removeChild(span2)
