WireShark USB

There are two types of USB interrept one is - Keyboard , Mouse

For keyboard you usually have Leftover Capture Data, you can give a command like this to get Leftover Capture Data (usb.capdata) and then store in a text file

tshark -r capture.pcapng -Y "usb.transfer_type==0x01 and frame.len==72 and frame.len!=64" -T fields -e usb.capdata -e frame.len > text2

Convert the data into desired format and then get mapping

Now you can follow this article: https://res260.medium.com/usb-pcap-forensics-barcode-scanner-nsec-ctf-2021-writeup-part-1-3-b0a5392c9313

You can also use tools in resources folder

For mouse you will be given HID data and they give you offsets of X and Y movement , get data like 

tshark -r capture.pcapng -Y "usb.transfer_type==0x01 and frame.len==71" -T fields -e usbhid.data > data

You may get data like the article https://res260.medium.com/usb-pcap-forensics-graphics-tablet-nsec-ctf-2021-writeup-part-2-3-9c6265ca4c40
Or like this in my case 

[('00', 'ff', 'ff', 'ff', 'ff'), ('00', 'ff', 'ff', 'ff', 'ff'),........

here i[0] = click done or not , i[2] = negative sign or not for x , i[4] = negative sign or not for y

The corresponding code is :

from PIL import Image

mouse_events = [('00', 'ff', 'ff', 'ff', 'ff'), ......]
img = Image.new('RGB', (10000, 10000), color='white')
canvas = img.load()
mouse_x = 5000
mouse_y = 5000

for data in mouse_events:
    left_button_pressed = int(data[0],16) & 0b00000001
    if data[2]=='ff':
        mouse_x -= 256-int(data[1],16)
    if data[2]=='00':
        mouse_x += int(data[1],16)
    if data[4]=='ff':
        mouse_y -= 256-int(data[3],16)
    if data[4]=='00':
        mouse_y += int(data[3],16)

    if left_button_pressed:
        # These two for loops are to make the pixels thicker
        for i in range(5):
            for j in range(5):
                canvas[round(mouse_x) + i, round(mouse_y) + j] = (0, 0, 0)

img.save("final.png")
